# 🎓 Student Management System

## 📌 Project Overview

A console-based Student Management System built using Java, JPA, and PostgreSQL.
This application allows users to manage student records efficiently with full CRUD operations.

---

## 🚀 Features

* ➕ Add new student
* 📋 View all students
* 🔍 Search student by ID
* ✏️ Update student details
* ❌ Delete student record
* 🗄️ Database integration using PostgreSQL

---

## 🛠️ Tech Stack

* **Language:** Java
* **Framework:** JPA (Java Persistence API)
* **Database:** PostgreSQL
* **Build Tool:** Maven

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/SimranVerma16/student-management-system.git
```

### 2️⃣ Open project

* Import as Maven project in IDE (Eclipse / IntelliJ)

### 3️⃣ Configure Database

* Install PostgreSQL
* Create a database (e.g., `student_db`)
* Update database credentials in `application.properties`

### 4️⃣ Run the project

* Run main class:

```
StudentManagementApplication.java
```
